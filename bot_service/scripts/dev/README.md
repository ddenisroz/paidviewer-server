# Скрипты `bot_service/scripts/dev`

Здесь лежат вспомогательные отладочные скрипты. Это не ежедневный рабочий слой и не часть нормального runtime.

## Что сюда относится

- проверка токенов
- проверка сессий
- проверка whitelist/admin-состояния
- локальные вспомогательные утилиты для диагностики
- одноразовые сценарии, которые ещё могут пригодиться разработчику

## Что сюда не относится

- рабочие maintenance-команды
- destructive reset/clear сценарии
- release cleanup

## Правило

Если скрипт нужен редко, зависит от локального окружения или предназначен в основном для отладки, он должен жить здесь, а не в корне `bot_service/scripts/`.
